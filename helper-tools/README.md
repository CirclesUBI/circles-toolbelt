# Helper tools

Helper tools for Circles development.

## Usage

### Install dependencies

```
npm install
```

### Generate .env

```
cp .env.example .env
```

### Trust safe

```
npm run trust-safe 0x405CFc20350E09d6514494934FF73dAEE3bAc461
```

### Fund safe

```
node ./fund-safe.js 0x405CFc20350E09d6514494934FF73dAEE3bAc461
```
